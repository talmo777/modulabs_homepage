프로젝트 진행 방식

체계적인 프로세스로 아이디어를 실제 해결책으로 만들어냅니다.

01

문제 정의

해결할 사회 문제를 명확하게 정의하고 범위를 설정합니다.

02

데이터 수집

공공 데이터 포털, API 등을 통해 필요한 데이터를 수집·정제합니다.

03

모델링 · 분석

통계·머신러닝 모델을 구축하고 패턴과 인사이트를 도출합니다.

04

제안 · 공유

결과를 보고서로 정리하고, 관련 기관에 정책 대안을 제안합니다.
